# Rust Lambda HTTP Example

A RUST Lambda Http Server with `cargo-lambda` and `lambda_runtime`

## Prepare
```sh
cargo install cargo-lambda ## need cargo-lambda
```

## Try
```sh
make run
make test-get
make test-post
```

## Deploy 
```sh
  make deploy
  make get # try remote function
  make post
```

## Helps
```
  make help
```
